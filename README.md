# Проверка шаблона ВКР

Данный репозиторий содержит исходники фронтенда сервиса для проверки шаблона ВКР.

---

## Технологии

- React TypeScript
- SCSS
- Vite

---

## Запуск и развертывание

- `npm run dev` - запуск для разработки
- `npm run build` - собирает проект в директорию build

После сборки проекта, его нужно добавить в директорию `/static` в бэкенде
